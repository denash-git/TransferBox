# 📦 TransferBox v3.0.0

[![OS Support](https://img.shields.io/badge/OS-Debian%2012%20%7C%2013%20%7C%20Ubuntu%2022.04%20%7C%2024.04-blue.svg)](https://debian.org)
[![Version](https://img.shields.io/badge/version-3.0.0-green.svg)](#)

**TransferBox** — это современная и удобная панель управления прокси-сервером на базе связки **Caddy** и **sing-box** с маскировкой под веб-сайт. Все протоколы работают на едином веб-порту **443** (HTTPS), а посторонние запросы перенаправляются на маскировочный сайт с защитой от сканирования.

---

## ✨ Возможности панели

*   **Поддерживаемые протоколы и модули:**
    *   VLESS (WebSocket, gRPC, XHTTP)
    *   NaiveProxy
    *   Mieru (mita)
    *   NetBird
*   **Интерактивный TUI-интерфейс:** Удобное меню управления пользователями, протоколами и подписками в терминале (управление стрелками клавиатуры).
*   **Встроенная диагностика и мониторинг:** Быстрый просмотр системных логов, мониторинг ресурсов сервера в реальном времени и тест скорости.
*   **Резервное копирование:** Создание бэкапов и восстановление настроек в один клик.
*   **Безопасное обновление:** Проверка и обновление панели напрямую из меню настроек с автоматической защитой от сбоев при установке.

---

## 💻 Требования

*   **ОС:** Чистая Debian 12 / 13 или Ubuntu 22.04 / 24.04 LTS с root-доступом
*   **Домен:** Доменное имя должно быть направлено (DNS A-запись) на IP-адрес вашего сервера
*   **Порты:** Свободные порты `80` и `443`

---

## 🚀 Установка и запуск

### 1. Установка на сервере
Выполните команду в консоли вашего сервера:
```bash
bash <(wget -qO- https://raw.githubusercontent.com/denash-git/TransferBox/main/install.sh)
```

### 2. Запуск меню управления
После успешного завершения установки запустите интерфейс управления командой:
```bash
menu
```

## 🛠️ Особенности для пользователя

*   **Автоматический SSL:** Выпуск и продление SSL-сертификатов Let's Encrypt для вашего домена происходят полностью автоматически.
*   **Закрытый фаервол:** Панель автоматически настраивает UFW фаервол, оставляя открытыми только порты для работы прокси и SSH.
*   **Ссылки подписок:** Для каждого пользователя генерируются универсальные ссылки на автообновляемые подписки и QR-коды для легкого импорта в клиенты.
