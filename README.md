# TransferBox

TransferBox — минималистичная панель управления прокси-сервером на базе **Caddy** и **sing-box** с маскировкой под веб-сайт. На порту 443 работают протоколы **NaiveProxy** и **VLESS** (WS, gRPC, XHTTP), а все посторонние запросы перенаправляются на маскировочный сайт с защитой от активного сканирования.

---

## Требования

*   **ОС:** Debian 12 / 13 (Ubuntu 22.04 / 24.04) с root-доступом
*   **Домен:** Доменное имя должно быть направлено (DNS A-запись) на IP вашего сервера
*   **Порты:** Свободные порты `80` и `443`

---

## Установка

Запустите установщик на сервере одной командой:

```bash
bash <(wget -qO- https://raw.githubusercontent.com/denash-git/TransferBox/main/install.sh)
```

---

## Использование

После завершения установки запустите интерактивное меню управления:

```bash
menu
```

---

## Технические детали

*   **SSL:** Сертификаты Let's Encrypt выпускаются Caddy автоматически.
*   **Бэкапы:** Бэкап пользователей и конфигурации сохраняется в `/opt/transferbox/backups/`.
*   **Фаервол:** Во время установки автоматически настраивается UFW фаервол (открыты порты SSH, 80/tcp, 443/tcp, 443/udp, все остальные порты закрыты).
